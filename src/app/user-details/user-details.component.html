Create an angular material table with the following instructions:

  • You have been given an Angular setup with all the necessary dependencies already installed.
  • Implement an Angular Material table that features pagination with 10 elements per page and sorting functionality.
  • Utilize the displayedColumns data provided in the project's JSON file (displayedColumns.json).
  • Create a data source for the table using the generator class(bookGenerator.ts).
  • Integrate a search box at the top of the table and enable filtering of table data based on the search string, all while preserving the sorting and pagination.
  • Ensure that the search box and the table are contained within a single <div>, and that this container is centered both vertically and horizontally.
